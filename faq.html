FAQ

Question 1:

How do I access the books on the database?

Answer 1:

A user has to be logged in in order to access the books in the database. The user can then search for books in the "Search Textbooks" section.

Question 2:

Can I add or delete books from the database? Can I edit the entries?

Answer 2:

Yes, but a user has to be a database administrator. Only authorized personnel are able to add or delete books from the database. The same goes for editing existing books in the database.

Question 3:

How do I look for specific books?

Answer 3:

Books can be searched for in a number of different ways. The most basic would be a broad search where the user simply clicks the search button in the "Search Textbooks section. Another way to search for books is specificially by title, course number, semester, author, and more through the drop-down menu underneath the search bar. The check box below the search parameters, when checked, only returns books that match what has been typed into the search bar exactly.

Question 4:

Do I need an account? Is it free?

Answer 4:

Yes, in order to access the database, you must first have an account linked to our site. At the moment there is no way to register a new account and you must have an account already to log in. There will be a feature in the future that will allow users to register a new account by linking an email and making a password. Registering a new account (or having one) is free.

Question 5:

Do I need to log out when I'm done?

Answer 5:

Yes. Please log out when you are done using our database. The site will remember your login and keep you logged in eve when you navigate to another page. Someone could potentially use the database posed as you. This could potentially be a liability for both the user and our system. Please be safe and logout when you are done. Thank you!
